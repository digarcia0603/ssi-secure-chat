# SSI - Sistema de Comunicação Segura

Projeto desenvolvido no âmbito da unidade curricular de Segurança de Sistemas Informáticos da Licenciatura em Engenharia Informática da Universidade do Minho.

## Descrição

Este projeto consiste no desenvolvimento de um sistema de comunicação segura entre clientes, com suporte a autenticação, troca segura de chaves, cifragem de mensagens e validação de certificados.

O sistema foi implementado com uma arquitetura cliente-servidor, garantindo confidencialidade, integridade e autenticação nas comunicações.
